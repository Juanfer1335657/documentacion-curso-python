# Print

## ¿Qué es `print` en Python?

En Python, `print()` es una función incorporada que se usa para
mostrar información en la pantalla (la salida estándar).
Es como decirle al programa: “escribe esto para que el usuario lo vea”.

## Como usar `print`

## 1. Mostrar texto

### Entrada:

```python title="Python"
print("Hola, mundo")
```

### Salida:

```python title="Python"
print("Hola, mundo")
```

## 2. Mostrar números

### Entrada:

```python title="Python"
print(5)
print(10 + 20)
```

### Salida:

```python title="Python"
5
30
```
